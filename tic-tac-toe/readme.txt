Tic Tac Toe Game
This is a simple implementation of the classic Tic Tac Toe game in [Programming Language/ Framework used].

Description
Tic Tac Toe is a two-player game where one player is assigned 'X' and the other 'O'. The game is played on a 3x3 grid. The players take turns to mark their respective symbol in an empty cell within the grid. The first player to get three of their symbols in a row (horizontally, vertically, or diagonally) wins the game. If all cells are filled and no player has achieved a winning combination, the game is a draw.

Features
Player vs. Player: Play against a friend on the same device.
Simple Interface: A user-friendly interface for an easy gaming experience.
How to Play
The game starts by displaying an empty 3x3 grid.
Player 1 (usually 'X') takes the first turn by clicking on an empty cell.
Player 2 (usually 'O') then takes their turn.
Players continue taking turns until a player wins or the game ends in a draw.
